# Color Picker Examples Demo

Comprehensive examples showing all color picker variants, sizes, and features.

## Files

- `color-picker-examples.php` → Copy to `app/Livewire/ColorPickerExamples.php`
- `color-picker-examples.blade.php` → Copy to `resources/views/livewire/color-picker-examples.blade.php`

## Usage

```blade
<livewire:color-picker-examples />
```

## Features Demonstrated

- Basic usage with labels
- Size variants (sm, default, lg)
- Style variants (outline, filled)
- Custom preset colors
- Standalone usage without label

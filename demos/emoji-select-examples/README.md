# Emoji Select Examples Demo

Comprehensive examples showing all emoji select variants, sizes, and features.

## Files

- `emoji-select-examples.php` → Copy to `app/Livewire/EmojiSelectExamples.php`
- `emoji-select-examples.blade.php` → Copy to `resources/views/livewire/emoji-select-examples.blade.php`

## Usage

```blade
<livewire:emoji-select-examples />
```

## Features Demonstrated

- Basic usage with search and category navigation
- With label integration
- Size variants (sm, default, lg)
- Style variants (outline, filled)
- Preselected emoji value
- In form groups
- Custom placeholder text
- Disabled search mode

## Requirements

- `wishborn/fancy-flux` package installed
- `livewire/flux` package installed
- Livewire component namespace configured

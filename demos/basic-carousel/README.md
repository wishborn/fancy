# Basic Carousel Demo

Simplest usage with data-driven slides. Perfect for image galleries or simple content carousels.

## Files

- `basic-carousel.php` → Copy to `app/Livewire/BasicCarousel.php`
- `basic-carousel.blade.php` → Copy to `resources/views/livewire/basic-carousel.blade.php`

## Usage

```blade
<livewire:basic-carousel />
```

## Features

- Data-driven slides via `:data` prop
- Auto-generated panels, controls, and steps
- Autoplay support
- Simple one-line implementation

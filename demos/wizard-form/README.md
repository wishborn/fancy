# Wizard Form Demo

Multi-step form wizard with validation and submission. Includes step tracking and success modal.

## Files

- `wizard-form.php` → Copy to `app/Livewire/WizardForm.php`
- `wizard-form.blade.php` → Copy to `resources/views/livewire/wizard-form.blade.php`

## Usage

```blade
<livewire:wizard-form />
```

## Features

- Step-by-step form collection
- Form validation
- Success modal on completion
- Step position restoration after Livewire re-renders
- `wire:submit` integration
